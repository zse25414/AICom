# Dogfood：5 分鐘新人上手（P1-1）

目標：全新使用者（可不設 DeepSeek Key）在 **5 分鐘內** 走完「建任務 → 教練帶做 → 勾選完成」。

## 前置

| 項目 | 命令／檢查 |
|------|------------|
| 全服務 | `npm run dev:all` |
| 就緒 | `curl http://127.0.0.1:3001/ready` 或 `npm run test:ready` |
| 前端 | 瀏覽器開 `http://127.0.0.1:3456/lumina-ai.html`（勿雙擊檔案） |

## 計時腳本（手動）

| # | 步驟 | 預期 | 上限 |
|---|------|------|------|
| 1 | 開啟頁面，可略過登入（訪客） | 看到「今日」與一句話價值主張 | 30s |
| 2 | 點 **一鍵體驗** 或輸入任務按 Enter | 任務出現在今日清單 | 60s |
| 3 | 點 **教練帶我做**（或進教練頁開始引導） | 出現步驟與教練訊息 | 60s |
| 4 | 回覆「完成這步」或點選項 | 進度前進或給出下一步 | 90s |
| 5 | **完成這件**／勾選任務 | 任務 completed，有回饋 | 60s |
| 6 |（可選）設定 → AI 設定嚮導 | 可不做也能完成 1–5 | — |

**通過標準**：1–5 合計 ≤ 5 分鐘，且無白屏／永久「思考中」。

## 離線／無 Key 檢查

- [ ] 未填 API Key 時教練仍有結構化回覆（非空白）
- [ ] 回覆含進度感與可點選項
- [ ] 首次可出現「連上 AI」提示，但**不強制**擋主路徑

## 工程自動化對照

| 路徑 | 命令 |
|------|------|
| 個人主路徑（jsdom） | `npm run test:e2e` |
| 階段 smoke | `npm test` |
| 團隊 API 主路徑 | `npm run test:e2e-team`（需 API 運行） |

## 失敗時

1. 開 Console 是否有 chunk 載入失敗 → 硬重新整理  
2. `GET /ready` 是否 ok  
3. 設定 → **回報問題** 複製診斷全文  
4. 見 `docs/engineering/RUNBOOK.md`
